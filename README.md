# DormFight - 寝室风云

一款基于 Python + Pygame 开发的拳皇风格格斗游戏。

## 游戏特色

- 4个可选择角色，每个角色有独特的属性和必杀技
- 完整的格斗系统：轻攻击、重攻击、必杀技、防御、连击
- AI 对战模式
- 双人对战模式
- 程序化生成的矢量风格角色精灵

## 角色介绍

| 角色 | 类型 | 特点 |
|------|------|------|
| 大力 | 力量型 | 高攻击、高防御、速度慢 |
| 快手 | 速度型 | 高速度、多连击、伤害低 |
| 全能 | 均衡型 | 各项能力平衡 |
| 技巧 | 技巧型 | 擅长中距离牵制 |

## 操作说明

### 玩家1（WASD + JKL）
- `W` - 跳跃
- `A/D` - 左右移动
- `S` - 下蹲
- `J` - 轻攻击
- `K` - 重攻击
- `L` - 必杀技（需要能量满）
- `U` - 防御

### 玩家2（方向键 + 小键盘）
- `↑` - 跳跃
- `←/→` - 左右移动
- `↓` - 下蹲
- `Num 1` - 轻攻击
- `Num 2` - 重攻击
- `Num 3` - 必杀技
- `Num 0` - 防御

## 安装与运行

```bash
# 安装依赖
pip install pygame

# 运行游戏
python main.py
```

## 游戏规则

- 每回合 99 秒
- 先赢得 2 回合的玩家获胜
- 血量归零或时间耗尽时回合结束
- 时间耗尽时，血量高的玩家获胜

## 技术栈

- Python 3.x
- Pygame

## 开发说明

游戏使用程序化生成的矢量风格角色精灵，无需外部图片资源。
如需添加真实音效和背景音乐，请在 `assets/sounds/` 和 `assets/music/` 目录添加对应文件。
