## 如何替换角色/小兵/武器

### 替换主角精灵

1. 准备 **96×63px 水平排列的 spritesheet**，PNG 格式
2. 文件命名：`idle.png` `walk.png` `jump.png` `punch.png` `kick.png` `jab.png` `hurt.png` `dive_kick.png` `jump_kick.png`
3. 放入对应目录：
   - 龚大哥(p1) → `assets/sprites/characters/p1/`
   - 军师(p2) → `assets/sprites/characters/p2/`
   - 神秘人(p3) → `assets/sprites/characters/p3/`
   - 籽桐(p4) → `assets/sprites/characters/p4/`
4. 重启游戏自动加载

### 替换小兵精灵

1. 准备 **96×63px 水平排列的 spritesheet**
2. 文件命名：`boxer_idle.png` `boxer_walk.png` `boxer_attack.png` `boxer_hurt.png` 等
3. 放入 `assets/sprites/minions/`

### 调整角色配色

修改 `characters/character_base.py` 中的 `CharacterStats`：

```python
color=(220, 50, 50),         # 主色
secondary_color=(255, 100, 100) # 次色
```

### 换武器类型

修改 `characters/character_base.py` 中的 `weapon_type`：

```python
weapon_type="fist"        # 拳套
weapon_type="pistol"      # 手枪
weapon_type="laser"       # 激光枪
weapon_type="shuriken"   # 手里剑
weapon_type="eagle"      # 白头海雕
weapon_type="dagger"     # 匕首
weapon_type="rifle"      # 突击步枪
weapon_type="nunchaku"   # 双截棍
weapon_type="flag"       # 红旗
```
